
# Challenge of the Day 2.0 (Эмпатичный Челлендж)

## Описание
Это не просто трекер задач, а приложение-компаньон, которое адаптируется под ваше эмоциональное состояние. Проект разработан как пример "умного" минималистичного интерфейса на Java (React/TS в данной реализации для веба), где логика сервисов имитирует архитектуру Spring-приложений.

### Ключевые концепции:
1. **MoodAnalyzer (Анализатор состояния)**: Анализирует ответы за прошлый день (энергия, настроение) и подбирает соответствующую категорию челленджа (Rest, Active, Productivity).
2. **Mystery Reveal (Тайна смысла)**: Некоторые задания кажутся простыми или странными, пока вы их не выполните. Объяснение пользы скрыто до момента завершения.
3. **Silent Days (Тихие дни)**: Система иногда отключает все счетчики и категории, предлагая просто "быть", снижая цифровое давление.
4. **Narrative Reports (Текстовые отчеты)**: Вместо графиков и очков — короткие истории о вашем прогрессе.

## Архитектура (Java Style)
- **ChallengeService/MoodAnalyzer**: Бизнес-логика выбора контента.
- **StorageService**: Уровень доступа к данным (DAO).
- **UI Components**: Чистые функциональные компоненты с акцентом на UX.

## Инструкция по запуску
- Откройте `index.html` в браузере. Приложение полностью автономно и сохраняет данные в LocalStorage.
